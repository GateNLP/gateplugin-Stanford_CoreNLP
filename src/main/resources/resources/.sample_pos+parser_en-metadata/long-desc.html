Parse text using the Stanford Parser. This application produces the following output annotations:

  • Token annotations with part-of-speech tags
  • Sentence annotations
  • SyntaxTreeNode annotations encoding the phrase structure tree, in a form compatible with the syntax tree viewer in the GATE Developer "Tools" plugin
  • Dependency annotations encoding dependency-style parse information.

This application uses the standard ANNIE POS tagger to generate the POS tags, and the parser uses these tags rather than generating its own, making the parsing process more efficient.
